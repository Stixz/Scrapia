# Scrapia
A small, scrappy utility from the Ravenforge toolkit.

Scrapia is a lightweight helper designed to automate tiny tasks, clean up small messes, and support your workflow without fuss.  
It’s simple, fast, and unpretentious — exactly what a scrappy tool should be.

## Features
- Quick, focused functionality
- Minimal configuration
- Built for everyday use
- Scrapia has a very unique personality

## Philosophy
Not every tool needs to be grand.  
Scrapia is a pocket‑knife: small, handy, and always ready.
